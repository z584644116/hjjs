# 标准条款原文库

本目录存放项目中引用的国家标准 / 行业标准 / 规程的条款原文与结构化数据。
所有代码里的阈值、限值、允差、公式参数都应来自此目录,不得硬编码 magic number。

## 目录结构

```
src/standards/
  jjg196-2006.ts     玻璃量器检定规程
  hj836-2017.ts      固定污染源废气 低浓度颗粒物 重量法
  hj493-2009.ts      水质采样 样品的保存和管理(未启用)
  gb13223-2011.ts    火电厂大气污染物排放标准(未启用)
  ...
```

## 每个文件的标准字段

```typescript
export const XXX = {
  id: '标准编号',
  fullName: '标准全名',
  version: '发布年',
  url: '可选 - 现行有效性查询链接',
  clauses: {
    '条款号': {
      pageRef: 'PDF p.X',
      rawText: '原文',
      extracted: {
        /* 结构化取值 */
      },
    },
  },
} as const;
```

## 来源置信度

每条 `extracted` 值在代码中使用时,应当带 `source` 字段说明:
- `standard` - 直接来自本条款原文,最高置信
- `empirical` - 行业经验,非标准强制
- `vendor`   - 厂家样本
- `inferred` - 推断/间接推导

## 核对流程

1. 新增/修改任何数值前,必须附 PDF 截图或 OCR 原文
2. PR 描述中引用原文段落
3. 无原文引用的数字改动不允许合并
