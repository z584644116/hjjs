# 环境计算工具模块公式与设计说明

- 生成时间：2026-04-16
- 项目：env-calculator
- 范围：当前代码中可见的全部计算模块（首页模块 + V23 聚合模块）

## 1. 全局设计

### 1.1 页面与组件结构
- 全局壳层：`TopNavigation + CalculatorShell + ResultDisplay + NumberInput`。
- 首页：`NavigationGrid`，按分类展示模块入口，支持搜索与分类筛选。
- 计算页：每个模块采用输入区 + 结果区模式，移动端保留底部操作栏。

### 1.2 通用输入与交互
- 数值输入组件会自动清洗字符（支持 `-` 与小数点）。
- 多数页面使用 `useMemo` 实时计算结果，少数页面通过“开始计算”触发。
- 结果通过 `ResultDisplay` 统一展示，支持状态色（success/warning/danger）。

### 1.3 通用校验与修约
- 正数校验：`requirePositive`。
- 非负校验：`requireNonNegative`。
- 普通四舍五入：`roundTo`（Math.round + EPS）。
- 银行家舍入：部分模块使用 `roundHalfToEven`（四舍六入五成双）。

## 2. 首页主模块

## 2.1 采样嘴计算（`/calculator/sampling`）

### 核心公式
- 干烟气流速：`Vd = Vw × (1 - Xw/100)`。
- 流量换算：`Q(m³/s) = Q(L/min) / 60000`。
- 理论截面积：`A = Q / Vd`。
- 理论嘴径：`d(mm) = 2 × sqrt(A/π) × 1000`。
- 保护功率流量：`Q_protect = 0.85 × Q_max`。

### 推荐规则
- 满功率推荐：基于 `Q_max` 计算理论嘴径。
- 保护功率推荐：基于 `0.85×Q_max` 计算理论嘴径。
- 从嘴径库中选择“`<=` 理论值的最大口径”；若都大于理论值，则取最小口径。
- 普通颗粒物库：`[4.5, 6, 7, 8, 10, 12]`。
- 低浓度颗粒物库：`[4, 4.5, 5, 6, 7, 8, 10, 12, 14, 15, 16, 18, 20, 22, 24]`。

### 输入与校验
- 输入：仪器型号、采样类型、烟气流速、含湿量。
- 校验：必须选仪器；烟气流速 `> 0`；含湿量 `0~100`。

### 输出与页面设计
- 输出：干烟气流速、满功率推荐嘴径、保护功率推荐嘴径、可用嘴径列表。
- 页面设计：计算器 Tab + 仪器管理 Tab；无仪器时禁用计算按钮。

## 2.2 烟气折算计算（`/calculator/fluegas`）

### 核心公式
- `C_ref = C_measured × (21 - O2_ref) / (21 - O2_measured)`。

### 输入与校验
- 输入：实测浓度、基准氧含量、实测氧含量。
- 校验：浓度 `>= 0`；氧含量均在 `0~21`；`O2_measured < 21`。

### 修约
- 折算浓度：银行家舍入到 2 位小数。
- 折算系数：银行家舍入到 4 位小数。

### 输出与页面设计
- 输出：折算后浓度、折算系数、输入参数回显。
- 页面设计：顶部展示公式提示卡；结果区支持错误提示与明细文本。

## 2.3 气体单位换算（`/calculator/gas`）

### 核心公式
- 常量：`R = 0.082057338 L·atm/(mol·K)`。
- 温度：`T_K = T(℃) + 273.15`。
- 压力：`P_atm = P(kPa) / 101.325`。
- `ppm -> mg/m³`：`C = ppm × M × P_atm / (R × T_K)`。
- `mg/m³ -> ppm`：`ppm = C × (R × T_K) / (M × P_atm)`。

### 气体范围
- `SO2`、`NO`、`NO2`、`CO`、`NMHC`。
- `NMHC` 按碳计（ppmC ↔ mgC/m³），使用碳摩尔质量 `12.011`。

### 输出与页面设计
- 输出：换算结果、温度、压力、分子量。
- 页面设计：气体下拉 + 输入单位下拉 + 数值输入；结果区展示换算说明。

## 2.4 大气稳定度与监测适宜度（`/calculator/unorg`）

### 核心计算链
- 太阳赤纬：按年积日 `dayOfYear` 与三角级数计算。
- 太阳高度角：
  `h0 = asin( sin(lat)sin(dec) + cos(lat)cos(dec)cos(15t + lon - 300) )`。
- 辐射等级：由 `h0 + 总云量 + 低云量` 分段判定（`-2 ~ 3`）。
- 稳定度级别：按“风速 × 辐射等级”查表（A~F，含过渡级 A-B/B-C）。
- 自定义高度风速换算：`u10 = uh × (10/h)^α`，`α` 由地表类型与稳定度决定。

### 适宜度规则
- 风向标准差（样本标准差）判级：
  `<15 => a`，`<30 => b`，`<=45 => c`，`>45 => d`。
- 风速判级：
  `<1.0 => d`，`<=2.0 => a`，`<=3.0 => b`，`<=4.5 => c`，`>4.5 => d`。
- 稳定度判级：`E/F=>a`，`D=>b`，`C/B-C=>c`，其余=>d。
- 综合级别：取三个子项中“最差级别”。
- 取消建议：存在 `d` 或 `c` 项达到 2 个及以上。

### 输入与页面设计
- 输入：日期、时间、经纬度、云量、风速方式、地表类型、10次风向风速序列。
- 页面设计：支持 GPS 一键填充经纬度；分别展示稳定度结果卡与适宜度结论卡。

## 2.5 溶解氧计算（`/calculator/do`）

### 核心公式
- 1 atm 饱和 DO：由 `0~40℃` 标准表 + 线性插值得到 `G2`。
- 水蒸气压：
  `es = 0.61121 × exp((18.678 - T/234.5) × T/(T + 257.14))`。
- 压力修正：
  `DO_raw = G2 × (P - es) / (101.325 - es)`。
- 标准值：`DO_std = roundHalfToEven(DO_raw, 2)`。
- 核查范围：`[DO_std - 0.5, DO_std + 0.5]`。

### 输入与校验
- 输入：大气压（kPa）、温度（℃）。
- 校验：温度范围 `0~40℃`。

### 输出与页面设计
- 输出：饱和溶解氧标准值、核查上下限。
- 页面设计：输入卡 + 结果卡，详情中固定提示“标准值 ±0.5 mg/L”。

## 2.6 pH 标准值（`/calculator/ph`）

### 核心模型
- 缓冲液：`1.68 / 4.00 / 6.86 / 9.18 / 12.46`。
- 数据策略：
  使用分段三次 Hermite（PCHIP）插值生成参考值，再在临近表点中择优匹配标准值。
- 另提供理论多项式 `theoreticalPH`（用于参考）。

### 输入与校验
- 输入：温度。
- 校验：总温度范围 `0~95℃`；`12.46` 缓冲液有效范围 `0~60℃`。

### 输出与页面设计
- 输出：各缓冲液标准值（不可用项显示“不可用”）。
- 页面设计：单输入 + 多结果列表，按缓冲液逐条展示。

## 2.7 地下水井水体积计算（`/calculator/well`）

### 核心公式
- 埋深：`buriedDepth = D - E`。
- 井水深度：`waterDepth = B + E - D`。
- 井管面积：`A_pipe = C²×π/4`。
- 扩孔面积：`A_enlarge = F²×π/4`。
- 环隙面积：`A_annulus = max(A_enlarge - A_pipe, 0)`。
- 高度换算：`height_cm = 100 × waterDepth`。
- 体积：
  `V(L) = (A_pipe×height_cm + A_annulus×height_cm×G) / 1000`。
- 该模块按 Excel 逻辑使用 `π = 3.14`。

### 规则与修约
- `G`（孔隙度）为空时，体积输出留空。
- 埋深/井水深度：银行家舍入 2 位小数。
- 体积：银行家舍入 1 位小数。

### 页面设计
- 输入：`B/C/D/E/F/G` 六项。
- 输出：埋深、井水深度、井水体积。

## 2.8 水质质量控制分析（`/calculator/wqc`）

### 模块说明
- 本页未拆到 `src/lib`，计算逻辑集中在页面内 `useMemo`。
- 由 6 个子分析组成：离子平衡、TDS校核、EC校核、硬度校核、溶解平衡、综合展示。

### 子分析 1：阴阳离子平衡
- 基础离子当量浓度（meq/L）
  `meq = 浓度(mg/L) / 当量重`。
- 基础当量重：K 39.1，Na 23.0，Ca 20.04，Mg 12.15，Cl 35.45，SO4 48.03，HCO3 61.02，CO3 30.005。
- 可选离子：`meq = 浓度 / (摩尔质量/|电荷数|)`。
- 误差：`error% = ((ΣC - ΣA)/(ΣC + ΣA)) × 100`。
- 判定：`|error| <= 10%` 合格。

### 子分析 2：TDS 与离子总量
- 离子总量：对全部离子浓度求和；`HCO3` 按 `×(60/122)` 换算计入 TDS。
- 误差：`(TDS_calc / TDS_measured - 1) × 100`。
- 判定：`|error| <= 10%` 合格。

### 子分析 3：TDS 与电导率
- 比值：`ratio = TDS / EC`。
- 判定区间：`0.55 ~ 0.70`。

### 子分析 4：电导率与离子校核
- 阳离子误差：`errorC = (ΣC×100/EC - 1) × 100`。
- 阴离子误差：`errorA = (ΣA×100/EC - 1) × 100`。
- 判定：两项都 `|error| <= 10%` 才合格。

### 子分析 5：总硬度核查
- 公式：`Hardness_calc = ((Ca/20)+(Mg/12)+(Fe3+/18.6)+(Mn2+/27.5)) × 50`。
- 误差：`(Hardness_calc / Hardness_measured - 1) × 100`。
- 判定：`|error| <= 10%` 合格。

### 子分析 6：沉淀溶解平衡（IAP/Ksp）
- 先将 mg/L 转为 mol/L。
- `IAP_CaCO3 = [Ca2+][CO3 2-]`。
- `IAP_CaSO4 = [Ca2+][SO4 2-]`。
- `IAP_PbCrO4 = [Pb2+][CrO4 2-]`。
- `IAP_PbSO4 = [Pb2+][SO4 2-]`。
- Ksp：CaCO3 `3.8e-9`，CaSO4 `2.4e-5`，PbCrO4 `1.8e-14`，PbSO4 `1.7e-8`。
- 判定：`IAP/Ksp > 1.1` 过饱和，`<0.9` 未饱和，其余饱和，`IAP=0` 数据不足。

### 页面设计
- 动态可增删“可选阳离子/阴离子”行。
- 结果区按 6 张 `ResultDisplay` 分块输出，支持状态色与明细。

## 3. V23 聚合模块（`/calculator/v23?module=...`）

## 3.1 标准状况采样体积（`air-volume`）
- 公式：
  `V_actual = Q × t`（L）
  `V_standard = V_actual × (P/101.325) × (273.15/(273.15+T))`（L）
- 输入：Q(L/min)、t(min)、P(kPa)、T(℃)。
- 校验：Q/t/P 必须 >0；温度必须 > -273.15℃。
- 输出：实际体积、标准体积。

## 3.2 环境空气 PM 浓度（`pm-conc`）
- 公式：
  `m = w2 - w1`（mg）
  `ρ = m / V0`（mg/m³）
- 输入：采样后滤膜重、采样前滤膜重、标准体积。
- 校验：体积 >0；`w2 >= w1`。
- 输出：颗粒物质量、质量浓度。

## 3.3 固定源颗粒物浓度（`stack-pm`）
- 公式：`C = (m / Vnd) × 1000`（mg/m³，Vnd单位 L）。
- 输入：捕集质量 m（mg）、标准干烟气体积 Vnd（L）。
- 校验：m 非负，Vnd >0。
- 输出：颗粒物浓度。

## 3.4 烟气等速跟踪率核查（`gas`）
- 绝对压力：`P_abs = Ba + Ps`。
- 密度估算：`ρs = 1.293 × (P_abs/101.325) × (273.15/(273.15+ts))`。
- 烟气流速：`Vs = Kp × sqrt(2×Pd/ρs)`。
- 理论流量：`Q_theory = Vs × A_nozzle × 60 × 1000`（L/min）。
- 偏差：`dev% = (Q_actual - Q_theory)/Q_theory × 100`。
- 判定：`|dev| <= 10%`。

## 3.5 AQI 空气质量指数（`aqi`）
- 公式：线性插值
  `IAQI = ((IAQI_HI-IAQI_LO)/(BP_HI-BP_LO)) × (C-BP_LO) + IAQI_LO`。
- 污染物：PM2.5、PM10、SO2、NO2、CO、O3(8h)。
- 判定等级：
  `<=50 一级优`，`<=100 二级良`，`<=150 三级轻度`，`<=200 四级中度`，`<=300 五级重度`，其余六级严重。

## 3.6 碱度/硬度换算（`alkalinity-hardness`）
- 公式：`mg/L(as CaCO3) = (V - V0) × C × 50000 / Vs`。
- 输入：滴定体积、空白体积、滴定液浓度、样品体积。
- 校验：`V>=0`，`V0>=0`，`C>0`，`Vs>0`，且 `V>=V0`。
- 输出：净滴定体积、换算结果。

## 3.7 MDL 方法检出限（`mdl-qc`）
- 公式：`MDL = t × s`。
- 输入：标准差 `s`、t 值。
- 校验：`s>=0`，`t>0`。
- 输出：MDL（与 s 同单位）。

## 3.8 加标回收率（`recovery-qc`）
- 回收量：`Recovered = C_spiked - C_original`。
- 回收率：`Recovery% = Recovered / Spike × 100`。
- 输入：原样值、加标样值、加标量。
- 校验：加标量 >0。
- 输出：回收量、回收率。

## 3.9 RSD 相对标准偏差（`rsd-qc`）
- 平均值：`mean = Σxi/n`。
- 样本标准差：`sd = sqrt(Σ(xi-mean)^2/(n-1))`。
- RSD：`RSD% = sd/mean × 100`。
- 校验：至少 2 个有效值；平均值不能为 0。
- 输出：平均值、标准偏差、RSD。

## 3.10 土壤含水率（`soil-moisture`）
- 水分质量：`m_water = m_wet - m_dry`。
- 含水率：`W% = m_water / m_dry × 100`。
- 校验：湿重、干重均 >0，且湿重 >= 干重。
- 输出：水分质量、含水率。

## 3.11 土壤制备质控（`soil-prep-qc`）
- 损失量：`loss = before - after`。
- 损失率：`loss% = loss / before × 100`。
- 过筛率：`sieve% = passed / total × 100`。
- 校验：`before>0`，`after>=0`，`after<=before`，`total>0`，`passed>=0`，`passed<=total`。
- 输出：损失量、损失率、过筛率。

## 3.12 统计样本量估算（`sample-size`）
- 公式：`n = (t × s / E)^2`。
- 输入：标准差 s、允许误差 E、置信系数 t（默认1.96）。
- 校验：三者均 >0。
- 输出：原始样本数（小数）与建议样本数（向上取整）。

## 3.13 污染物超标倍数（`exceedance`）
- 标准指数：`SI = C / L`。
- 超标倍数：`Exceed = max(0, SI - 1)`。
- 判定：`SI > 1` 视为超标。
- 输入：实测浓度 C、标准限值 L。
- 校验：`C>=0`，`L>0`。
- 输出：标准指数、超标倍数、是否超标。

## 4. 结果展示与判定风格（具体设计）

- 所有模块统一使用结果列表卡片，便于现场快速核查。
- 判定结果采用状态色：
  success 对应“合格/通过”，warning 对应“临界/关注”，danger 对应“不合格/超限”。
- 大部分模块边输边算；对输入前置条件较多的模块（如采样嘴、无组织监测）使用显式“开始计算”。
- 对“可解释性”要求高的模块提供 details 区，展示公式解释或中间量。
