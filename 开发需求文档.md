# 环境检测计算器 v2.3 公式单位补充说明

> **文档版本**：v2.3（单位核实版）  
> **更新日期**：2026-04-14  
> **适用项目**：基于 Next.js 15.5.3 + React 19.1.0 + TypeScript + Turbopack + Fluent UI React Components 9.70.0 + Zustand 5.0.8  
> **核心目标**：对新增及扩展公式的计量单位进行严格核实与补充，确保计算结果符合国家标准文件规定，消除用户单位混淆风险。

---

## 一、项目概览

环境检测计算器是一款面向环境监测现场/实验室的专业即时计算工具，支持空气/固定源、水质、土壤/地下水、通用 QC 等**20+ 模块**。全部公式均来自生态环境部官方标准 PDF 原文（HJ/T 397-2007、HJ 194-2017、HJ 618-2011、HJ 836-2017、HJ 633-2012、HJ 168-2010 等），纯客户端 TS 实现，**完全离线可用**。

**本次更新重点**：  
- 对 v2.3 新增的 14 个公式模块进行**计量单位补全与核实**。  
- 明确每个输入参数的合法单位及输出结果的单位格式。  
- 提供开发时用于控件 suffix、tooltip 提示的标准单位字符串。

## 三、新公式模块计量单位补充（经标准文件核实）

依据各 HJ/GB 标准原文，逐一对新增及扩展公式的**输入参数单位、输出结果单位**进行严格定义。

### 1. 标准状况下采样体积换算 (`/calculator/air-volume`)

| 符号 | 含义 | 单位（规范写法） | 输入/输出控件建议 |
| :--- | :--- | :--- | :--- |
| `Q` | 采样器工作流量 | **L/min** | suffix="L/min" |
| `t` | 累计采样时间 | **min** | suffix="min" |
| `P` | 采样时大气压 | **kPa** | suffix="kPa", step=0.1 |
| `t_s` | 采样时环境温度 | **°C** | suffix="°C", step=0.1 |
| `V_n` | **标准状况采样体积** | **L** | 结果单位 **L** |

**核实依据**：HJ 194-2017 附录 A。

### 2. 环境空气 PM10/PM2.5 质量浓度 (`/calculator/pm-conc`)

| 符号 | 含义 | 单位 | 控件设计 |
| :--- | :--- | :--- | :--- |
| `w_2` | 采样后滤膜重量 | **mg** | suffix="mg", step=0.01 |
| `w_1` | 采样前滤膜重量 | **mg** | suffix="mg", step=0.01 |
| `V_0` | 标准状况采样体积 | **m³** | suffix="m³", step=0.1 |
| `ρ` | **颗粒物质量浓度** | **mg/m³** | 结果单位 **mg/m³**（公式乘 1000 后即得） |

### 3. 固定污染源颗粒物浓度 (`/calculator/stack-pm`)

| 符号 | 含义 | 单位 | 核实细节 |
| :--- | :--- | :--- | :--- |
| `m` | 捕集颗粒物质量 | **mg** | HJ 836-2017 规定滤筒增重以 mg 计 |
| `V_nd` | 标准干烟气采样体积 | **L** | 公式乘 10⁶ 隐含单位为 L，输入控件后缀明确为 **L** |
| `C` | **颗粒物浓度** | **mg/m³** | 结果单位 |

### 4. 烟气流速与等速采样验证 (`/calculator/gas` 扩展)

| 参数 | 含义 | 单位 | 备注 |
| :--- | :--- | :--- | :--- |
| `P_d` | 动压 | **Pa** | |
| `B_a` | 大气压 | **kPa** | 与 P_s 统一单位 |
| `P_s` | 静压 | **kPa** | 负压时输入负值 |
| `t_s` | 烟气温度 | **°C** | |
| `ρ_s` | 烟气密度（可计算） | **kg/m³** | |
| `K_p` | 皮托管系数 | 无量纲 | 默认 0.84 |
| `V_s` | **烟气流速** | **m/s** | 结果保留 2 位小数 |
| `Q_实际` | 实际采样流量 | **L/min** | |
| `Q_理论` | 等速要求流量 | **L/min** | |

### 5. 空气质量指数 AQI (`/calculator/aqi`)

- **输入**：SO₂、NO₂、O₃ 为 **μg/m³**；CO 为 **mg/m³**（依据 HJ 633-2012）。
- **UI 策略**：根据污染物选择动态切换输入框单位 suffix。

### 6. 方法检出限 MDL (`/calculator/mdl-qc`)

| 符号 | 含义 | 单位 | 规范 |
| :--- | :--- | :--- | :--- |
| `s` | 重复测定标准差 | 与浓度单位一致 | **mg/L**, **μg/m³** 等 |
| `t` | t 分布值 | 无量纲 | 通常 n=7 时 t=3.143 |
| `MDL` | **方法检出限** | **与输入单位一致** | |

### 7-11. 其余新增模块单位速查表

| 模块 | 输入/输出关键单位 | 核实依据 |
| :--- | :--- | :--- |
| **加标回收率与 RSD** | 加标量 **mg/L** 或 **μg**；回收率 **%**；RSD **%** | HJ 168-2010 |
| **滴定法碱度/硬度** | 滴定体积 **mL**；浓度 **mol/L**；结果 **mg/L (以 CaCO₃ 计)** | GB/T 11893 |
| **土壤含水率** | 湿重/干重 **g**；含水率 **%** | HJ/T 166-2004 |
| **土壤样品制备损失率/过筛率** | 重量 **g**；损失率/过筛率 **%** | HJ 系列 2024 |
| **采样方案统计样本数** | 标准差、允许误差单位与总体一致 | GB 12997-91 |
| **污染物超标倍数** | 浓度 **mg/L** 或 **μg/m³**；倍数 **无量纲** | HJ 663 |

## 四、文件结构建议（公式层）

为保持公式逻辑独立可测，建议以下目录结构：
/lib/calculators/
├── air-volume.ts # Vn 换算
├── pm-conc.ts # PM 浓度
├── stack-pm.ts # 固定源颗粒物
├── gas.ts # 烟气（含等速验证）
├── aqi.ts # AQI 六参数
├── mdl-qc.ts # 检出限
├── recovery-qc.ts # 加标回收与 RSD
├── alkalinity-hardness.ts # 碱度/硬度
├── soil-moisture.ts # 土壤含水率
├── soil-prep-qc.ts # 土壤制备损失/过筛
├── sample-size.ts # 统计样本数
└── exceedance.ts # 超标倍数

text

每个模块文件包含纯函数计算逻辑，通过 JSDoc 标注公式来源与单位假设。
