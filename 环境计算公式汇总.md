# 环境计算器 — 项目技术文档

> 本文件汇总项目的技术架构、UI 布局设计及全部计算公式。公式参数与单位均与源码保持一致。

---

## 一、项目概览

环境计算器（Environment Calculator）是一款面向环境监测领域的专业计算工具，支持多种常见计算模块的前端运行，采用浏览器本地存储实现数据持久化，适用于本地开发和云端部署。

### 存储架构

- **客户端存储**：浏览器 `localStorage`，通过 `StorageAdapter` 接口统一封装
- **Zustand 持久化**：状态变更自动同步至 localStorage，无需手动管理
- **用户数据隔离**：注册用户数据以 `env_calc_user_data_${username}` 为键前缀存储
- **服务端 API**（可选）：`/api/storage` 提供服务端 CRUD 操作，`/api/health` 提供健康检查

### 部署拓扑

    │
    ▼

    │
    ▼
Next.js 应用（/api/health、/api/storage）
```

```
┌─────────────────────────────────────────────┐
│           TopNavigation（顶部导航栏）          │
│  [Logo/标题]           [用户认证] [仪器管理]  │
├─────────────────────────────────────────────┤
│                                             │
│              内容区域（各计算模块页面）          │
│                                             │
└─────────────────────────────────────────────┘
```

### 首页导航（NavigationGrid）

首页以卡片网格形式展示各计算模块入口，布局特征：

- **网格布局**：响应式多列网格（`auto-fit` + `minmax`），桌面端多列、移动端单列
- **卡片风格**：每个模块一张卡片，包含图标、标题和简要描述
- **交互反馈**：悬停态变化，点击进入对应计算页面

### 计算页面通用布局

各计算模块页面遵循统一布局模式：

```
┌─────────────────────────────────────────────┐
│ Breadcrumb（面包屑导航）                      │
│ 首页 > [模块名称]                             │
├─────────────────────────────────────────────┤

### 认证模块

- **AuthSection**：顶部导航栏右侧的用户认证入口（Guest / 注册切换）
- **AuthModal**：弹窗模式，支持登录、注册、恢复密钥操作
- **Recovery Key**：注册时生成的 UUID 恢复密钥，用于密码重置

### 响应式策略

- 栅格系统依赖 Fluent UI 的响应式 tokens
- 输入区域：`grid-template-columns: repeat(auto-fit, minmax(180px, 1fr))`
- 移动端自适应：网格列数随视口宽度自动调整

### 设计风格

- **风格**：简约专业，符合环境监测行业工具的审美
- **色彩**：以中性灰为基础色，Fluent UI 主题色为点缀
- **字体**：Fluent UI 默认字体栈，标题用 `Title1/Title2`，正文用 `Body1`

## 1. 采样嘴计算 (`calculator.ts`)

### 干烟气流速

$$
V_d = V_w \times (1 - X_w)
$$

- \(V_d\)：干烟气流速（m/s）
- \(V_w\)：烟气流速（m/s）
- \(X_w\)：烟气含湿量（%）

### 理论嘴径计算

由流量方程 \(Q = A \times v\) 推导：

$$
d = 2 \times \sqrt{\frac{Q}{\pi \times v}}
$$

- \(Q\)：采样流量（L/min），转换为 m³/s 时除以 60000
- \(v\)：干烟气流速（m/s）
- \(d\)：计算直径（mm），结果再从标准直径库中选择 ≤ 计算值的最大者

### 保护功率

保护功率系数 = 85%，即流量按满功率的 85% 代入公式重新计算推荐嘴径。

## 2. 溶解氧饱和度计算 (`do.ts`)

### 饱和 DO 查表（1 atm，0–40 °C）

使用标准查表表（DO_TABLE_1ATM），整数温度直接查表；非整数温度在相邻整数间线性插值：

$$
DO(T) = DO_{floor} + (DO_{ceil} - DO_{floor}) \times \frac{T - T_{floor}}{T_{ceil} - T_{floor}}
$$

### 水汽压校正（Tetens 公式）

$$
e_s = 0.61121 \times \exp\left( \frac{(18.678 - T/234.5) \times T}{T + 257.14} \right)
$$

- \(e_s\)：水面饱和水汽压（kPa）
- \(T\)：水温（°C）

### 气压校正

$$
DO_{校正} = DO_{1atm} \times \frac{P - e_s}{101.325 - e_s}
$$

- \(P\)：实际大气压（kPa）
- 结果四舍六入五成双（Banker's rounding）保留 2 位小数

## 3. pH 标准值计算 (`ph.ts`)

### 理论 pH（多项式拟合）

各缓冲溶液在 25 °C 下的理论 pH 用温度多项式计算：

| 缓冲溶液 | 多项式公式 |
|---------|-----------|
| pH 1.68 | \(1.6679 + 0.0002T + 0.00001T^2 + 0.00000001T^3\) |
| pH 4.00 | \(4.0057 - 0.0016105T + 0.00007005T^2 - 0.0000004T^3 + 0.99\times10^{-12}T^4\) |
| pH 6.86 | \(6.9802 - 0.0067T + 0.00009T^2 - 3.07375\times10^{-7}T^3\) |
| pH 9.18 | \(9.4405 - 0.0113T + 0.000053T^2\) |
| pH 12.46 | \(13.416 - 0.0415T + 0.0001028T^2 + 0.000001055T^3 - 7.1\times10^{-9}T^4\) |

### 查表值选取

使用分段三次 Hermite 插值（PCHIP）在标准表数据点间插值，再在 [T−10, T+10] °C 范围内选取与理论值最接近的表值。最终显示值四舍六入五成双保留 2 位小数。

## 4. 水质 QC 分析 (`wqc/page.tsx`)

### 阴阳离子平衡

各离子毫当量浓度：

$$
\text{阳离子 meq/L} = \sum \frac{C_i}{M_i / |z_i|}
$$

其中 \(C_i\) 为浓度（mg/L），\(M_i\) 为摩尔质量（g/mol），\(z_i\) 为离子价数。

**阴阳离子总浓度：**
$$
C_{cat} = \frac{K^+}{39.1} + \frac{Na^+}{23.0} + \frac{Ca^{2+}}{20.04} + \frac{Mg^{2+}}{12.15}
$$
$$
C_{an} = \frac{Cl^-}{35.45} + \frac{SO_4^{2-}}{48.03} + \frac{HCO_3^-}{61.02} + \frac{CO_3^{2-}}{30.005}
$$

**相对误差：**
$$
误差(\%) = \frac{C_{cat} - C_{an}}{C_{cat} + C_{an}} \times 100
$$

合格标准：|误差| ≤ 10%

### TDS 计算

$$
TDS_{计算} = K^+ + Na^+ + Ca^{2+} + Mg^{2+} + Cl^- + SO_4^{2-} + CO_3^{2-} + HCO_3^- \times \frac{60}{122}
$$

合格标准：|相对误差| ≤ 10%

### TDS / 电导率比值

$$
r = \frac{TDS}{EC}
$$

合格范围：0.55 ≤ r ≤ 0.70

### 总硬度（mg/L as CaCO₃）

$$
硬度 = \left( \frac{Ca^{2+}}{20.04} + \frac{Mg^{2+}}{12.15} + \frac{Fe^{3+}}{18.6} + \frac{Mn^{2+}}{27.5} \right) \times 50
$$

合格标准：|相对误差| ≤ 10%

### 沉淀溶解平衡

离子活度积：
$$
IAP = [M^{z+}] \times [X^{z-}]
$$

判定规则：

| 条件 | 状态 |
|------|------|
| IAP / Ksp > 1.1 | 过饱和 |
| IAP / Ksp < 0.9 | 未饱和 |
| 0.9 ≤ IAP / Ksp ≤ 1.1 | 饱和 |

Ksp 参考值：

| 物质 | Ksp |
|------|-----|
| CaCO₃ | 3.8 × 10⁻⁹ |
| CaSO₄ | 2.4 × 10⁻⁵ |
| PbCrO₄ | 1.8 × 10⁻¹⁴ |
| PbSO₄ | 1.7 × 10⁻⁸ |

## 5. 无组织排放监测适宜度 (`unorg.ts`)

### 太阳高度角

$$
h_0 = \arcsin(\sin\phi \sin\delta + \cos\phi \cos\delta \cos t)
$$

- \(\phi\)：纬度
- \(\delta\)：太阳赤纬（度）
- \(t\)：时角（度）

### 太阳赤纬

$$
\delta = 0.006918 - 0.399912\cos Q + 0.070257\sin Q - 0.006758\cos 2Q + 0.00907\sin 2Q \\
- 0.002697\cos 3Q + 0.00148\sin 3Q
$$
其中 \(Q = \frac{360 \times d}{365}\)（度），\(d\) 为年内日期序号。

### 大气稳定度等级

根据 10m 风速和辐射等级（−2 ~ 3），查表确定稳定度类别（A / A-B / B / B-C / C / D / E / F）。

### 10m 风速换算

$$
u_{10} = u_h \times \left( \frac{10}{h} \right)^\alpha
$$

- \(\alpha\)：风速指数（按城市/乡村和稳定度等级查表）
- \(h\)：实际测量高度（m）

### 适宜度评定

| 指标 | 评定规则 |
|------|---------|
| 风向标准差 | σθ < 15° → a；< 30° → b；≤ 45° → c；> 45° → d |
| 平均风速 | < 1.0 m/s → d；≤ 2.0 → a；≤ 3.0 → b；≤ 4.5 → c；> 4.5 → d |
| 稳定度 | E/F → a；D → b；B-C/C → c；A/A-B → d |

最终结果取最差（worst）等级。监测取消条件：存在任意 1 个 d 类，或 2 个及以上 c 类。

## 6. 气体流速/浓度换算 (`gas.ts`)

### ppm ↔ mg/m³ 换算

通用气体状态方程：
$$
PV = nRT
$$

$$
C_{mg/m^3} = C_{ppm} \times \frac{M \times P}{R \times T}
$$
$$
C_{ppm} = C_{mg/m^3} \times \frac{R \times T}{M \times P}
$$

- \(M\)：气体摩尔质量（g/mol）
- \(P\)：大气压（kPa）
- \(T\)：温度（K）
- \(R = 0.082057338\) L·atm/(mol·K)

支持气体：SO₂、NO、NO₂、CO、NMHC（非甲烷总烃，以碳计）。

## 7. 烟气折算浓度 (`fluegas.ts`)

### 基准氧含量折算

$$
C_{ref} = C_{meas} \times \frac{21 - O_{2,ref}}{21 - O_{2,meas}}
$$

- \(C_{ref}\)：折算后污染物浓度
- \(O_{2,ref}\)：基准氧含量（%）
- \(O_{2,meas}\)：实测氧含量（%）

合格条件：实测 O₂ < 21%（分母为正）。结果四舍六入五成双保留 2 位小数。

## 8. 地下水井设计 (`well.ts`)

### 基础参数

$$
埋深 = D - E
$$
$$
井水深度 = B + E - D
$$

- \(B\)：井建深度（m）
- \(D\)：地下水位埋深（m）
- \(E\)：井口高出地面高度（m）

### 井水体积（L）

$$
V_{井水} = \frac{V_{套管} + V_{填砾}}{1000}
$$

其中（π 取 3.14）：

$$
V_{套管} = \frac{\pi C^2}{4} \times 100 \times H_{water}
$$
$$
V_{填砾} = \left( \frac{\pi F^2}{4} - \frac{\pi C^2}{4} \right) \times 100 \times H_{water} \times G
$$

- \(C\)：套管内径（cm）
- \(F\)：扩孔直径（cm）
- \(G\)：填砾孔隙度（0–1）
- \(H_{water} = B + E - D\)（水柱高，m）

埋深、水深保留 2 位小数，体积保留 1 位小数，全部使用四舍六入五成双。

## 附：通用修约规则

项目中所有涉及"四舍六入五成双"（Banker's Rounding）的计算，均遵循以下规则：

```
若小数部分恰好等于 0.5：
  → 舍入到最接近的偶数（0, 2, 4, 6, 8 结尾）
否则：
  → 按常规四舍五入
```

涉及修约的模块：DO、pH、采样嘴、烟气折算、井设计。
